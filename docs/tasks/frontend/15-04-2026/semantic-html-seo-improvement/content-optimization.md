# Content Optimization

## Target Halaman

- `/`
- `/login`
- `/register`

## Fokus Keyword

Landing page diarahkan ke intent pencarian seperti:

- aplikasi catat keuangan keluarga
- catat pemasukan dan pengeluaran
- aplikasi keuangan keluarga
- scan struk dan pencatatan transaksi

## Perubahan Konten

### Landing Page

- title dan description diselaraskan dengan intent pencarian Indonesia
- hero image diberi alt yang mendeskripsikan manfaat utama halaman
- navigation landing diubah menjadi anchor links nyata ke section penting

### Login dan Register

- title dan description dibuat jelas sesuai fungsi halaman
- tetap publik untuk akses user, tetapi tidak ditargetkan untuk indexing organic search

## Internal Linking

- link CTA ke `/register` dan `/login` tetap dipertahankan
- menu landing kini memakai anchor links ke section `Tentang`, `Fitur`, `Pricing`, dan `FAQ`

## URL Structure

- `/` untuk landing publik
- `/login` untuk autentikasi user existing
- `/register` untuk pendaftaran user baru

Struktur ini tetap ringkas, konsisten, dan mudah dipahami crawler maupun user.
